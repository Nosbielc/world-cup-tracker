# Copa Tracker

Site estático para acompanhar uma competição de futebol no estilo Copa do Mundo, com foco em experiência mobile, placar ao vivo simulado, próximos jogos, grupos e chaveamento eliminatório.

## Arquivos principais

- `index.html`: estrutura da aplicação
- `styles.css`: layout responsivo, tema claro/escuro e animações
- `data.js`: dados simulados de times, grupos, jogos, eventos, classificação e chaveamento
- `app.js`: renderização, filtros, interações e simulação do jogo ao vivo

## Como executar localmente

Como o projeto usa apenas HTML, CSS e JavaScript puro, basta abrir o arquivo `index.html` em qualquer navegador moderno.

## Publicando no GitHub Pages

1. Faça push da branch com os arquivos do projeto.
2. No GitHub, abra **Settings** do repositório.
3. Entre em **Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch desejada e a pasta `/ (root)`.
6. Salve a configuração e aguarde a publicação.

Depois disso, o site ficará disponível pela URL do GitHub Pages do repositório.
